# Generador de Contratos PDF

Aplicación React para generar contratos profesionales en PDF con tres tipos disponibles: prestación de servicios, arrendamiento y colaboración entre autónomos.

## Características

- **3 tipos de contrato**: Servicios, Arrendamiento, Colaboración
- **Campos dinámicos**: Formulario adaptable según el tipo seleccionado
- **Vista previa en tiempo real**: Visualiza el contrato antes de descargar
- **Generación automática**: Número de contrato y fecha automáticos
- **Descarga PDF**: Genera PDFs profesionales y listos para usar
- **Diseño responsive**: Funciona perfecto en mobile y desktop

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Uso

1. Selecciona el tipo de contrato
2. Completa el formulario con los datos de ambas partes
3. Añade los campos específicos del tipo de contrato
4. Visualiza el contrato en la vista previa
5. Descarga el PDF

## Stack Tecnológico

- React 18 + Vite
- TypeScript
- Tailwind CSS
- jsPDF + html2canvas
- Lucide React
